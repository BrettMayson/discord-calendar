# Discord Calendar Bot

Create a public calendar for Google Calendar or your favorite calendar service that shares your Discord events.

[Invite the bot to your server](https://discord.com/oauth2/authorize?client_id=1512651387035713616&scope=bot%20applications.commands&permissions=8589934592)
